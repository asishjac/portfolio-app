# Portfolio App

A personal portfolio application built with React and Vite.

## Features

- **Sidebar Navigation**: Easy access to different sections of the portfolio.
- **Main Content Area**: Displays the selected content dynamically.
- **Responsive Design**: Adapts to different screen sizes.

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Lucide React](https://lucide.dev/) (Icons)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/portfolio-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Usage

Run the development server:

```sh
npm run dev
```

Open http://localhost:5173 to view it in the browser.

## Deployment

This project is configured for deployment to GitHub Pages.

```sh
npm run deploy
```
